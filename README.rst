Vimmortal
=========

The vim conf that makes you invincible!

Installing
----------

::

    git clone git@github.com:teh-cmc/vimmortal.git
    make -C vimmortal install compile-ycm

That's it

Requirements
------------

- Vim 7.3.584+
- CMake 2.7+
